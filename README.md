# Online Food Ordering System

A web-based Online Food Ordering System developed using PHP and MySQL.  
This project allows users to browse restaurants, view menus, place orders, and track them online.  
Admins can manage restaurants, menus, and orders through a dashboard.

## Features
- User registration and login
- Restaurant and menu management
- Online food ordering
- Order tracking
- Admin dashboard

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## How to Run the Project
1. Install XAMPP
2. Copy the project folder to `htdocs`
3. Start Apache and MySQL
4. Import the database into phpMyAdmin
5. Open `http://localhost/1.onlinefood/` in your browser
